# 🚖 OLA Ride-Hailing Data Analysis

## Project Overview
This project analyzes OLA ride booking data using Python, SQL, and Power BI to identify booking trends, cancellations, customer behavior, and revenue insights.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SQL
- Git & GitHub
